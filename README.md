# DSA Daily Practice

Simple structure to practice DSA daily and track progress on GitHub.

## Folder Structure

- `daily-practice/` - Daily logs (what you solved each day)
- `topics/` - Problems grouped by topic
- `progress/` - Overall tracker and weekly summary

## Suggested Daily Flow

1. Pick 1-3 problems.
2. Solve and add code in the related `topics/` folder.
3. Add a short daily note in `daily-practice/`.
4. Update `progress/progress-log.md`.
5. Commit and push.

## Simple Commit Format

- `day: add 2 array problems`
- `day: solve binary tree traversals`
- `week: update progress log`
