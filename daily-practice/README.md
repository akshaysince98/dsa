# Daily Practice Logs

Create one markdown file per day using this format:

- `YYYY-MM-DD.md`

Example:

- `2026-03-01.md`

## Suggested Template

```md
# 2026-03-01

## Problems Solved
- [ ] Two Sum (Easy)
- [ ] Valid Parentheses (Easy)

## Time Spent
- 90 minutes

## Notes
- Key learning points...
```
